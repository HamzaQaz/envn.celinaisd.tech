ARCHITECTURE
-------------------------------------------------------------------

Domain Setup:
- Frontend: envn.celinaisd.tech
- API/Backend: enapi.celinaisd.tech

Request Flow:
Internet → DNS → Nginx (80/443) → 
  ├→ envn.celinaisd.tech → Static Files (client/dist/)
  └→ enapi.celinaisd.tech → Proxy → Backend (localhost:3000)
                                      ├→ Express API
                                      └→ Socket.io WebSocket
